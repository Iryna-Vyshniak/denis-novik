# Denis Novik — Personal Portfolio Landing Page

A highly optimized, fully responsive, and semantic single-page portfolio website built entirely with Vanilla HTML5 and Vanilla CSS.

This project is a pixel-perfect implementation of a modern UX/UI designer portfolio landing page inspired by a professional Figma layout. The architecture focuses on accessibility, semantic correctness, clean CSS structure, performance optimization, and scalable maintainability without relying on JavaScript frameworks or CSS utility libraries.

---

# ✨ Overview

The project demonstrates how a modern landing page can be built using only native web technologies while maintaining:

- clean semantic structure
- scalable CSS architecture
- optimized rendering performance
- responsive layouts
- accessibility best practices
- maintainable design systems

The landing page includes:

- Hero section
- About section
- Skills section
- Portfolio/projects showcase
- Contact/footer section
- Reusable SVG icon system
- CSS-only rating components
